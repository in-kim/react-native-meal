# react-native-meal
react native 학습용으로 구현한 음식 소개 앱

## Getting Started
1. xcode, android studio emulrator 실행
2. npm install
```bash
npm install
# or
yarn install
```
3. android, Ios 앱 실행
```bash
npm run start
# or
yarn start
```
위 명령어 실행 후 터미널 창에서 I(IOS), A(Android) 입력
